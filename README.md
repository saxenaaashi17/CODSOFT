# CodSoft Python Programming Internship

This repository contains the Python projects completed for my CodSoft Python Programming Internship.

## Task 1: To-Do List Application

A GUI-based To-Do List application built using Python and Tkinter.

### Features
- Add new tasks
- Mark tasks as completed
- Delete selected tasks
- Clear all tasks

### Run the project
```bash
python todo_list.py
```


## Task 3: Password Generator

A Python GUI application that generates secure random passwords.

### Features
- Choose password length
- Include uppercase and lowercase letters
- Include numbers and optional symbols
- Copy generated password to clipboard

### Run the project
```bash
python password_generator.py
```

## Task 5: Contact Book

A GUI-based Contact Book application built using Python and Tkinter.

### Features
- Add contacts
- View contact details
- Search contacts by name or phone number
- Update contact information
- Delete contacts

### Run the project
```bash
python contact_book.py
```


## Technologies Used
- Python
- Tkinter
- `secrets` and `string` modules
